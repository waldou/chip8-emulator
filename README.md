chip8-emulator
=================

My attempt to create a CHIP-8 emulator.

### Requirements

- Java 11.

### How to run

Run `Main.java` with first argument as the ROM file path.

### Status

- Fonts are not working.
- Missing sound.
- Missing unit tests.
